Receitas Online: Um Catálogo Digital de Sabores

O projeto "Receitas Online" é um site responsivo com receitas classificadas por tipo e dificuldade. Criado com HTML e CSS, o design inclui imagens atrativas e descrições detalhadas.

Os usuários podem explorar receitas divididas em categorias como sobremesas, almoços rápidos e pratos vegetarianos. A interface foi pensada para facilitar a navegação em dispositivos móveis.

Quiz Climático: Jogo Educativo sobre Mudanças no Planeta

Um quiz interativo desenvolvido com JavaScript que aborda temas como aquecimento global, poluição e energias renováveis. As perguntas se adaptam às respostas, promovendo uma experiência de aprendizado dinâmica.

O visual foi pensado para ser envolvente, com cores fortes e ilustrações relacionadas à natureza. O código também está estruturado em módulos para facilitar atualizações futuras.
